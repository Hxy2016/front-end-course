# TodoList
## 需求
做一个和 http://todomvc.com/examples/jquery/ 功能一样的待办事宜列表。其主要功能如下：
* 代办列表的增删改查。
* 刷新页面或下次打开浏览器时
  * 维持上次的查询条件
  * 代办列表的数据还在

## 任务分解
* 完成 UI 部分
* 设计待办列表的数据结构和需要定义的函数
* 实现新增功能
* 实现修改功能
* 实现删除功能
* 实现查询功能
* 完成刷新页面或下次打开浏览器时
  * 维持上次的查询条件
  * 代办列表的数据还在
